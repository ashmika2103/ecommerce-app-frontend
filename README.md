# E-Commerce Web Application - Frontend

## Live Demo

Frontend:
https://ecommerce-app-frontend-kappa-five.vercel.app

Backend API:
https://ecommerce-app-backend-8hnf.onrender.com

## Features

* User Registration
* User Login
* Product Listing
* Add to Cart
* Order Placement
* Order Tracking
* Responsive Navigation

## Technologies Used

* React.js
* React Router DOM
* Axios

## Installation

```bash
npm install
npm start
```

Runs on:
http://localhost:3000

## Deployment

Frontend deployed on Vercel.
